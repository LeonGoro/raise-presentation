## БАЗА ЗНАНИЙ: ТИПОГРАФИКА (из canva.com/ru_ru/obuchenie/tipografika/ —
## применять во ВСЕХ проектах Леона)

- Максимум 2–3 шрифта; лучше ОДНА гарнитура в разных начертаниях (SemiBold/Bold
  для заголовков, Regular для текста, Italic для подзаголовков).
- Кегль заголовка = кегль текста × 1.6 (золотое сечение). Крупный кегль = самое
  важное, средний = описание, мелкий = примечания.
- Интерлиньяж = 120–125% кегля. Чем длиннее строка — тем больше интерлиньяж.
- Правило близости: внутренние расстояния < внешних. Буквы<слова<строки<абзацы.
  Отступ заголовка СНИЗУ меньше, чем СВЕРХУ (заголовок липнет к своему тексту).
- Кернинг не трогать (в хороших шрифтах он оптимален).
- Длина строки: 45–75 знаков для кириллицы (узкие колонки 30–35).
- Выравнивание по левому краю (центр/право/ширина — хуже читается).
- Чёткая иерархия h1→h2→h3; взгляд должен скользить по структуре.
- Контраст текста и фона обязателен; тёмный на светлом или наоборот.
- Вдовы/сироты (висячие строки) — убирать. Тире «—» в предложении, дефис «-»
  в слове, между числами короткое тире «–» без пробелов. Кавычки «ёлочки»,
  внутри — „лапки“. Неразрывные пробелы: г. Омск, 50 кг, и т. д.
- Пустое пространство — инструмент: дизайн должен «дышать», не перегружать.
