# RAISE LLC - Interactive Presentation

Modern, responsive web presentation with 22 slides using React, TypeScript, and Framer Motion.

## Features
- 🎨 Beautiful gradient design
- ⌨️ Keyboard & scroll navigation
- 📱 Fully responsive
- ✨ Smooth animations
- 🎯 Click indicators for quick navigation

## Quick Start

```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy on Railway
Push to GitHub and Railway auto-deploys!
